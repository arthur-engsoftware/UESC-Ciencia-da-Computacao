## 🧩 Portugol — Fundamentos de Lógica de Programação

O **Portugol** é uma linguagem de pseudocódigo utilizada para ensinar lógica de programação antes da transição para linguagens compiladas. Os exercícios abaixo praticam os pilares do raciocínio algorítmico.

| Arquivo | Conceito Praticado | Descrição |
|---|---|---|
| `ola_mundo.por` | Estrutura de programa, saída de dados | Primeiro algoritmo: uso de `escreva` para exibir mensagem |
| `antecessor_sucessor.por` | Variáveis, operadores aritm., E/S | Leitura de inteiro e cálculo de antecessor/sucessor |
| `calculo_divisao.por` | Operadores aritméticos, divisão inteira vs. real | Diferença entre `div` (quociente) e `/` (resultado real) |
| `calculadorasimples.por` | **Estruturas de Controle** — `se/então/senão` | Calculadora com seleção de operação via `escolha/caso` |
| `parouimpar.por` | **Estruturas de Controle** — operador módulo `%` | Verificação de paridade via resto da divisão |

### Conceitos-Chave em Portugol

- **Variáveis e Tipos**: `inteiro`, `real`, `caractere`, `lógico`
- **Operadores**: aritméticos (`+`, `-`, `*`, `/`, `%`), relacionais (`=`, `<>`, `<`, `>`), lógicos (`e`, `ou`, `não`)
- **Estruturas de Controle de Seleção**: `se/então/senão`, `escolha/caso`
- **Entrada e Saída**: `leia` e `escreva`
